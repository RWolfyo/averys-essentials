Welcome to Winter's Tweaks, a fork of Avery's Essentials. This modpack includes all the features of the original Avery's Essentials with additional visual and sound enhancements, as well as some comfort mods.

You can download Winter's Tweaks via Modrinth.

## FAQ

- **Q** What does bleeding-edge mean?
- **A** _Bleeding-edge means that Winter's Tweaks has less testing on updates compared to other modpacks. This can make it less stable but more up-to-date._

- **Q** What does rolling release mean?
- **A** _A rolling release model means updates are released as soon as non-breaking changes are made, resulting in smaller, more frequent updates. This is different from stable releases, which are less frequent but more thoroughly tested._

- **Q** Can I build off of this pack?
- **A** _Yes, feel free to modify and build upon this pack._

- **Q** What is this project licensed under?
- **A** _It's MIT licensed._
